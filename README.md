# SecGuess
16 cards across 4 categories — Web Attacks, Network, Malware & AI (including Prompt Injection), and mixed. Each card has a real scenario, keyword chips, 4 multiple choice answers, a hint button (costs 30 pts), and a full explanation after answering.
Controls:

📱 Phone → tilt left/right or forward/back to change cards (asks for gyroscope permission on iOS)
💻 Desktop → arrow keys ← → ↑ ↓ or Prev/Next buttons
Tap any answer choice to lock it in

Scoring:

100 pts per correct answer (70 if hint used)
Streak bonus: +15 pts per consecutive correct answer (up to +75)
Score + streak counter visible in the HUD at all times

Modes: All Topics / Web Attacks / Network / Malware & AI — cards shuffle randomly each session.
